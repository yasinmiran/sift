**What matters today:** A day after Apple sued OpenAI over trade secret theft, the follow-on reporting turns to what it costs: [Mark Gurman traces the lawsuit to months of tension between OpenAI hardware chief Tang Tan and his old boss, former Apple hardware chief John Ternus](https://www.techmeme.com/260711/p8#a260711p8), while [M.G. Siegler argues the suit could sideline OpenAI's hardware ambitions for years, "don't poke the bear"](https://www.techmeme.com/260711/p9#a260711p9), even as [Benedict Evans reads the broader AI market as heading toward frontier models becoming commodity infrastructure once the token crunch eases](https://www.techmeme.com/260711/p11#a260711p11), with value moving to the products built on top. Elsewhere, a security researcher's teardown of [xAI's Grok Build CLI found it uploads entire local repositories to Google Cloud Storage as git bundles](https://gist.github.com/cereblab/dc9a40bc26120f4540e4e09b75ffb547), a ==27,800x gap between data uploaded and data the model actually processed==, regardless of whether "Improve the model" is switched off.

## AI / LLMs

- [Zvi Mowshowitz weighs in on "Plan A," the AI 2027 team's new positive-scenario proposal for slowing and steering AI development](https://thezvi.wordpress.com/2026/07/11/introduction-for-and-reactions-to-plan-a/), engaging the objections point by point rather than endorsing it outright; the same authors' AI 2027 predictions have held up well enough that their next forecast is worth watching even from skeptics.
- [Yael Grauer argues "ask an LLM" has become a polite brush-off](https://blog.yaelwrites.com/stop-telling-me-to-ask-an-llm/), pointing out she has usually already exhausted the chatbot route before asking a person for their specific, lived experience, something a model can't substitute.
- [Mesh LLM lets developers point any OpenAI-compatible client at a local endpoint and distribute inference across a peer mesh built on iroh](https://www.iroh.computer/blog/mesh-llm), splitting models too large for one machine into pipelined layer stages across 40+ catalog models from 0.5B to a 235B mixture-of-experts giant, MIT licensed.
- [Nvidia published a framework for evaluating general-purpose robot policies before real-world deployment](https://developer.nvidia.com/blog/how-to-evaluate-general-purpose-robot-policies-for-real-world-deployment/), arguing that as foundation models get better at following natural-language pick-and-place instructions, benchmarking how they generalize matters more than raw task success rate.

## Devtools & Infra

- [ClickHouse Managed Postgres quadrupled PgBouncer throughput](https://clickhouse.com/blog/pgbouncer-clickhouse-managed-postgres), from 87k to 336k transactions per second, by running a fleet of PgBouncer processes with SO_REUSEPORT instead of one, since PgBouncer itself is single-threaded and was leaving most cores idle.
- [Evan Hahn makes the case for SQLite's STRICT tables](https://evanhahn.com/prefer-strict-tables-in-sqlite/), which reject wrong-typed inserts (text into an INTEGER column) that SQLite normally allows silently, though there's still no ALTER path to strict, only copy-and-recreate. [Simon Willison's sqlite-utils 4.1 shipped the same week with a CLI flag to toggle that strict mode on transform](https://simonwillison.net/2026/Jul/11/sqlite-utils/#atom-everything), plus --code for inline Python row generators and automatic primary-key detection on upsert.
- [Ant is a new JavaScript runtime and ecosystem](https://antjs.org) bundling its own engine, a package manager, the ants.land registry, app hosting, and an Electron-alternative desktop shell into one coherent platform rather than a language spec plus bring-your-own-tooling.
- [Ship That Code teaches Redis, Git, and database internals by having learners rebuild each from scratch](https://shipthatcode.com), a choose, write, run loop through Go, SQL, caches, queues, and protocols rather than reading about how they work.

## Security & Privacy

- [A teardown of xAI's Grok Build CLI found it uploads entire local repositories, unread files included, to a Google Cloud Storage bucket as git bundles](https://gist.github.com/cereblab/dc9a40bc26120f4540e4e09b75ffb547): a 12GB test repo sent 5.1GB to storage while the model itself processed only 192KB, and disabling "Improve the model" doesn't stop it, the upload isn't surfaced in the CLI's setup materials.

- [Bending Spoons, which owns Vimeo, AOL, and Evernote, received 800,000 job applications last year and made 286 hires](https://www.techmeme.com/260711/p12#a260711p12), the Wall Street Journal reports, a sub-0.04% acceptance rate the piece frames as harder than getting into Harvard.

## Elsewhere

- [The FCC authorized Reflect Orbital's Eärendil-1, an 18x18 meter mylar mirror satellite, to test reflecting sunlight down to a roughly 5km-wide patch of ground on demand](https://theconversation.com/the-u-s-just-approved-a-giant-space-mirror-to-test-sunlight-on-demand-low-earth-orbit-is-getting-weird-283482), the first of a planned 50,000-satellite constellation; astronomers warn the full fleet could raise night-sky brightness 200-300% and risk temporary flash-blinding pilots and drivers.
